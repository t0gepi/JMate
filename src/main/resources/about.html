About JMate

An open source project to play online chess at lichess.org
If you have any ideas to improve this project, feel free to contact me.

How to use

  • Engine

    Choose the engine you want to use.
    Note: If you want to play chess variants, you will need to use FairyStockfish since it's the only engine that supports variants.
    Note: Selecting any other Engine than FairyStockfish will therefore set the variant to standard chess.

  • Variant

    Choose the variant you want to play.
    If you are unfamiliar with the variants, you can read more about them here.
    Note: Selecting any variant other than standard chess will set the engine to FairyStockfish.

  • Opening

    Choose the preferred opening you want to play.
    Note: when using autoplay, the opening will only be played if opponent plays theory
    Note: Arrows only display moves of the selected opening if opponent plays theory

  • Depth

    Set the depth of the engine.
    Note: The higher the depth, the longer the engine will take to find the best move.
    Tip: For low time control use a low depth and autoplay

  • Start

    When pressed, the bot will wait until a game is found and then play(if autoplay is enabled) with your settings.

  • Autoplay

    If enabled, the bot will do the moves automatically using the settings.

  • Arrows

    If enabled, the bot will show the best moves of the selected opening using arrows.

  • Hotkey

    Press this button and then select a key to act as a hotkey. When autoplay is disabled, pressing the hotkey will stop the bot from calculating.
    The Bot will then play the best move it has found so far. To unset a hotkey, press the button and click the delete key.
